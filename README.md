### Как все работает
При запуске mune.bat вам будет показанно главное меню вы можете выбрать
```
1. Запуск програм
2. Не откравать
3. Экстренное отколючение
4. Выйти
```
Если вы выберите Запуск программ то вам предложется список программ
```
1. Бот Рандом
2. Бот Угадай число
3. Назад
```
В моем случии это Бот Рандом и Бот  Угодай число вибеирите программу прграмма откроется в одельном окне там будет показаны ошибки и сообщение о перезапуске. Чу-чуть подробнее mune.bat открывает не программу а батник вмоем случии это random.bat и number.bat когда батник отроется то запуститься программа это сделанно для того чтобы при подении программы он автоматически перезапускался. Так же на главном меню mune.bat есть надпись 3. Экстренное отключение если вы вибирите эту кнопку то mune.bat убиет батники в моем случии это random.bat и number.bat и процессы остановятся. Так же при экстренном отключении mune.bat сперва скажет видите слово типо подтверждаете процесс и еще mune.bat создаст txt фаил в той папке где лежит mune.bat в txt файле бедет записоваться дата и время говорится типо Эстренное отключение.

Если вы выбирите Не открывать то mune.bat вам откровет рик-рол, я добавил это просто для прикола.
Если вы вибирите Выйти то mune.bat закроется

### Как все запустить ?
Для начало надо скачать батники и подкоректировать название батников mune.bat надо оставить а вот number.bat и random.bat можно по менять имя дальше в mune.bat в коде надо поменять название батников на которое вы поменяли я вам показал где надо понять.  
```
:bot1
start number.bat
goto run_bots

:bot2
start random.bat
goto run_bots
```
И еще чтобы для красоты надо поменять в коде mune.bat
```
echo ========================
echo      Запуск программы
echo ========================
echo 1. Бот Рандом
echo 2. Бот Угадай число
echo 3. Назад
echo ========================
```
И самое главное надо в программы положить в папку с mune.bat и указать имя запускаемого файла програма в моем случии запускает Python.
Вот где надо поменять название в батниках с вашим названием и там поменять запсук файла вот пример:
```
echo [INFO] Загрузка бота Угадай число... 
python you_bot1.py
echo [WARN] Бот Угадай число упал! Перезапуск через 5 секунд... 
```

Все программа настроенна и готова к работе запускаейте mune.bat и пользуйтесь!
